## Hakkında

**[Plise Perde](https://pliseperde.com)** — Türkiye'nin plise perde alanında en kapsamlı dijital platformu. 93+ sayfa, 25+ bileşen ve tam SEO optimizasyonu ile profesyonel bir web uygulaması.

[Plise Perde](https://pliseperde.com), pencere profili içine takılan, katlanır yapılı, ışık ve gizlilik kontrolü sağlayan modern bir perde sistemidir. Bu platform, plise perde hakkında bilmeniz gereken her şeyi tek bir yerde toplar:

- **Ürün çeşitleri** — Honeycomb, düet, blackout, desenli, baskılı modeller
- **Montaj sistemleri** — Vidalı, vidasız, yapışkanlı, kancalı, portray, braketli, motorlu
- **Kullanım alanları** — 22 farklı kullanım alanı (mutfak, salon, yatak odası, karavan, tekne, hastane, vb.)
- **Fiyat hesaplama** — Online interaktif fiyat hesaplama aracı
- **Enerji tasarrufu** — Honeycomb kumaş ile ısı yalıtımı ve enerji tasarrufu hesaplayıcı
- **Karşılaştırmalar** — Plise perde vs zebra, jaluzi, stor, tül perde karşılaştırmaları

- **UV Korumalı** — %95-99 UV filtreleme, mobilya renk koruması
- **Güneş Kırıcı** — Işık filtreleme, parlamasız aydınlatma
- **Işık Kontrollü** — Kademeli açma, 5 farklı açılım tipi
- **Nem Dayanımlı** — Poliester kumaş, küflenmez
- **Kolay Temizlenebilir** — Nemli bezle silme, yıkama gerekmez
- **Alev Geciktirici** — EN 13501-1 standardı, kendiliğinden sönme
- **Antibakteriyel** — Gümüş iyon kaplama, bakteri engelleme

## Başlangıç

### Gereksinimler

- Node.js 18.17+
- npm veya yarn

### Kurulum

```bash
# Repository'yi klonla
git clone https://github.com/PlisePerde/pliseperde.git
cd pliseperde

# Bağımlılıkları yükle
npm install

# Development server'ı başlat
npm run dev
```

Site [http://localhost:3000](http://localhost:3000) adresinde çalışır.

### Build

```bash
# Production build
npm run build

# Server'ı başlat
node server.js
```
